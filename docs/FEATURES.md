# Features — 核心功能模块

> 返回 [README](../README.md)

---

## 1. 仪表盘 (Dashboard)

平台首页，提供全局概览：

- **统计卡片** — 项目总数、工作流数量、运行次数、文件数量
- **最近项目** — 按更新时间排列，快速进入
- **最近运行** — 显示运行状态、耗时、关联工作流
- **最近文件** — 运行产生的文件产物

---

## 2. 项目管理 (Projects)

多项目隔离的工作流管理：

- 创建 / 重命名 / 删除 / 归档项目
- 搜索与排序（按名称、创建时间、更新时间）
- 项目详情页：包含工作流列表、运行历史、文件产物
- 从模板快速创建工作流

---

## 3. 工作流编辑器 (Workflow Editor)

基于 ReactFlow 的可视化节点编排：

### 节点角色 (8 种)
| 角色 | 说明 |
|------|------|
| Input | 工作流入口，接收外部输入 |
| Planner | 任务规划与拆解 |
| Worker | 核心执行节点 |
| Reviewer | 质量审查与反馈 |
| Research | 信息搜集与检索 |
| Router | 条件路由与分发 |
| Human | 人工介入节点 |
| Output | 工作流出口，汇总输出 |

### 编辑器功能
- 拖拽添加节点、连线编排
- 节点检查器 (RightInspector)：概览 / 职责 / 任务 / 状态 / 日志 / 配置 6 个 Tab
- 配置 Tab 内各区块可折叠：节点配置、执行模式、工具绑定、Skill 绑定、文档上传、消息发送
- 节点模板库 & 任务树视图
- 工作流保存状态实时反馈
- 工作流版本发布

### 节点配置
- Agent 名称 / 描述 / 职责 / System Prompt / Additional Prompt
- 模型配置：Provider / Model / API 凭证 / Temperature
- 工具策略：disabled / allowed / required
- 工具绑定 & Skill 绑定（运行级）
- 执行模式：标准 (LLM Agent) / 开发 (脚本执行) / 脚本节点 (命令模板)
- 人工输入开关

---

## 4. 运行中心 (Run Center)

全局运行管理与分析，分为两个独立视角：

### 工作流运行
- **统计卡片** — 总运行数、成功率、平均耗时、Token 总量
- **趋势折线图** — 7/30 天运行次数趋势
- **成功率饼图** — 成功 / 失败 / 运行中分布
- **Token 柱状图** — 各工作流 Token 消耗对比
- **节点耗时排行** — 横向条形图
- **节点失败率排行** — 找出瓶颈节点

### 开发运行
- 趋势折线图、成功率饼图、脚本统计
- 按脚本 / 命令维度分析

### 运行列表
- 搜索 / 排序 / 状态筛选
- 点击进入运行详情页

---

## 5. 运行详情 & 执行追踪 (Run Detail)

深度调试工作流执行过程：

- **Execution Timeline** — 甘特图式节点执行时间线，展示各节点的开始/结束/耗时
- **节点 I/O 查看** — 每个节点的输入输出结构化展示
- **Prompt Trace** — System Prompt / User Prompt / Message History / Completion / Token 计数
- **Tool Call Trace** — 工具名、入参 JSON、出参 JSON、耗时、错误信息
- **事件日志** — 完整的运行事件流

---

## 6. 开发台 (Agent Dev)

独立的 Agent 脚本开发环境：

### 工作台管理
- 创建 / 删除工作台
- 工作台列表展示：绑定路径、最近运行状态
- 最近运行卡片：显示命令、入口文件、耗时、退出码，点击可打开脚本

### IDE 环境
- **Monaco 编辑器** — 语法高亮、多 Tab 编辑
- **文件树** — 支持文件 / 文件夹的创建、删除、重命名；折叠状态跨会话保持
- **集成终端** — XTerm.js 终端
- **一键运行** — 自动检测当前文件类型，支持 14 种语言扩展名（.py / .js / .ts / .go / .rs 等）
- **运行追踪** — 每次运行自动创建记录，纳入运行中心

### 双模式
- **Workspace 模式** — 使用平台托管的工作区文件
- **本地项目模式** — 绑定本地文件夹，直接编辑本地文件

---

## 7. 资产管理 (Assets)

统一管理平台资源：

| 资产类型 | 说明 |
|----------|------|
| 工作流模板 | 预设工作流结构，一键应用 |
| 节点模板 | 可复用的节点配置 |
| 模型 | LLM Provider / Model 定义 |
| Prompt 模板 | 可复用的 Prompt 片段 |
| 工具 | 外部工具/函数定义与绑定 |
| 技能包 | Skill 资产，可绑定到节点 |
| 参考文档 | 参考资料，注入节点上下文 |

---

## 8. 全局功能

- **全局搜索** — 跨项目 / 工作流 / 运行 / 文件搜索
- **快速新建** — 顶栏 +新建 按钮，快速创建项目或工作流
- **通知中心** — 运行成功 / 失败通知
- **凭证管理** — API Key 等敏感凭证的安全存储
